# casa-automatizada-iot

Este projeto consiste no desenvolvimento de uma planta funcional de uma residência com cinco cômodos automatizados. O foco principal foi a integração de sistemas embarcados com tecnologias web e sustentabilidade.

Funcionalidades
- Controle Remoto via Web: Interface online para ligar e desligar a iluminação de cada cômodo individualmente.
- Eficiência Energética: O sistema inteiro opera alimentado por energia solar.
- Automação de Iluminação: Controle independente de LEDs em 5 ambientes distintos.

Tecnologias Utilizadas
- Microcontrolador (Arduino UNO)
- Linguagem C++ (Programação embarcada)
- HTML/CSS (Interface Web)
- Painel Solar
